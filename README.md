# DeepseekAPI

A dotnet wrapper for deepseek API, targeting framework on netstandard2.1, providing type-safe API calling in Unity or other programming framework that requires an older version of .NET environment.

## Quick start (Tests)

On linux:
```bash
git clone <repo-url>
export DEEPSEEK_API_KEY="<Your API-KEY here>"
cd DeepseekAPI.Tests
dotnet test
```

On windows (Powershell)
```ps1
git clone <repo-url>
$env:DEEPSEEK_API_KEY="<Your API-KEY here>"
cd DeepseekAPI.Tests
dotnet test
```
